# TabBar Icons

这个目录包含底部导航栏的图标文件。

## 需要的文件
- home.png - 首页图标（未选中）
- home-active.png - 首页图标（选中）
- classify.png - 分类图标（未选中）
- classify-active.png - 分类图标（选中）
- my.png - 我的图标（未选中）
- my-active.png - 我的图标（选中）

## 图标规格
- 尺寸：81px × 81px
- 格式：PNG
- 建议：使用透明背景

## 临时说明
当前使用占位符文件，需要替换为实际的图标文件。

## 如何替换
1. 准备6个PNG图标文件（81×81px）
2. 将文件复制到此目录
3. 重新编译项目
